# UI Competitive Roadmap

Missing enterprise UI flows that depend on the data layer. Prioritized for competitive parity with Pave, Carta, Ravio, Radford.

## High Priority (Core Differentiators)

### 1. Pay Bands Management
- CRUD bands by role/level/location (min, mid, max)
- Band width configuration (+/- % from midpoint)
- Auto-refresh bands based on market data
- Employee position-in-range visualization
- Band penetration analytics

### 2. Approval Workflows
- Multi-level approval chains (manager -> director -> HR -> exec)
- Threshold-based routing (e.g. >10% increase needs VP approval)
- Collaborator assignments with permission levels
- Approval status tracking and notifications
- Audit trail for all approvals

### 3. Exports & Scheduled Reports
- PDF report generation (benchmarks, pay equity, compliance)
- Excel export with formatting
- Scheduled report delivery (email, Slack)
- Custom report template builder

### 4. Granular Permissions
- Department/team-level access controls
- Manager scoping (cascading org hierarchy)
- Data field-level permissions
- Permission audit trail

### 5. Audit Log Viewer
- Comprehensive audit log (filtering, search, export)
- Change tracking for all compensation modifications
- User activity audit trail
- Data access audit logs

## Medium Priority

### 6. Pay Equity Statistical Analysis
- Regression-based pay gap analysis
- Multi-factor analysis (gender, ethnicity, tenure)
- Controlled vs uncontrolled gap breakdown
- Remediation cost modeling

### 7. Scheduled Reports
- Recurring report schedules
- Alert configuration dashboard

### 8. Benchmark Confidence Visualization
- Sample size indicators
- Data freshness badges (already in Data Health)

## Low Priority

- Predictive analytics dashboard
- Total rewards statements (from competitor gap plan)
- Interactive offer letter generation
